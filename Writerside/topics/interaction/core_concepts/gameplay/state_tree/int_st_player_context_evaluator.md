# Player Context Evaluator
<primary-label ref="interaction"/>

Resolves the **player controller** and **pawn** associated with the **active interaction**.

| Property            | Type    | Description                                          |
|---------------------|---------|------------------------------------------------------|
| Interaction Manager | Context | Interaction Manager assigned to the owner.           |
| Player Controller   | Output  | Player Controller participating in this interaction. |
| Player Pawn         | Output  | Player Pawn participating in this interaction.       |
