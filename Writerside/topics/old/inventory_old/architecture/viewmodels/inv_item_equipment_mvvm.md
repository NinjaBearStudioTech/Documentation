# Item Equipment
<primary-label ref="inventory"/>

Exposes information from the [**Equipment Fragment**](inv_equipment_fragment.md).

| Field               | Description                                    | Update Condition                   |
|---------------------|------------------------------------------------|------------------------------------|
| Has Equipment       | Informs if the item has an equipment instance. | Equipment is created or destroyed. |
| Equipment State Tag | Gameplay Tag representing the equipment state. | Equipment state changes.           |
