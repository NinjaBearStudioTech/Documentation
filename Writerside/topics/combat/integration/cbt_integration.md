# Integrations
<primary-label ref="combat"/>

Ninja Combat can be integrated with other plugins from Ninja Bear Studio, to further enhance its capabilities. This 
section will walk you through the **possibilities** and **implementation**.

**[Ninja Factions](cbt_integration_factions.md)**
: The **[Ninja Factions](fct_overview.md)** plugin can be used to automatically disable "friendly fire", between members 
of the same faction, with a friendly attitude towards each other.

**[Ninja Input](cbt_integration_input.md)**
: The **[Ninja Input](ipt_overview.md)** plugin can be used to properly trigger inputs in a way they are expected by the
Combat System, especially regarding the [Combo Manager](cbt_combo_system.md). Furthermore, the Input Buffer can be handy
for certain setups.

**[Ninja Inventory](cbt_integration_inventory.md)**
: The **[Ninja Inventory](inv_overview.md)** plugin can replace the Weapon Manager, allowing the Combat System to obtain
weapons directly from the [Inventory Manager](inv_inventory_manager.md).
