# Block Ability
<primary-label ref="combat"/>

<tldr>
    <ul>
        <li>Applies the <b>Block Gameplay Effect</b> that usually modifies attributes such as <b>Block Chance</b> and <b>Block Amount</b>.</li>
        <li>Can optionally check for a <b>cost</b>, such as <b>stamina cost</b> required to start blocking.</li>
        <li>Blocking events may trigger changes in the <b>Animation</b> and <b>Movement</b> modules.</li>
    </ul>
</tldr>
