# Combat Components
<primary-label ref="combat"/>

The Combat System uses multiple components dedicated to specific tasks.

This section will provide you a detailed breakdown of each component and how they should be used.

[Combo Manager](cbt_combo_manager.md)
: The Combo Manager manages multiple aspects of the <b>Combo System</b>: the <b>Combo Window</b>, <b>Combo Counter</b> 
and the <b>Combo State Tree</b>.

[Weapon Manager](cbt_weapon_manager.md)
: The Weapon Manager provides weapons that are currently available to the Combat System, to perform <b>Melee Scans</b>
or <b>Launch Projectiles</b>.
